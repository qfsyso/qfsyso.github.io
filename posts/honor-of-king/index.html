镜#

一、技能机制解析#

  1. 被动-铸镜
    攻击目标后触发镜像标记，再次攻击可触发额外伤害并刷新技能CD，刷野/连招时需穿插平A触发被动。
  2. 一技能-开锋
    突进并强化普攻，主要位移和起手技能，命中减少冷却时间。
  3. 二技能-裂空
    范围伤害+回血+减速，开局优先加点用于续航和快速清野。
  4. 三技能-见影
    定身敌人并展开镜像场，本体与镜像协同攻击，核心连招与换位收割技能。

二、出装与铭文推荐#

出装方案#

  • 核心装备：
    红色打野刀 → 抵抗之靴 → 暗影战斧 → 宗师之力 → 破军 → 碎星锤/名刀
    • 高爆发流：破军+碎星锤瞬秒脆皮
    • 生存调整：敌方控制多时换魔女斗篷，物理爆发高换反伤刺甲

铭文搭配#

  • 通用暴击流：
    10隐匿（移速/攻击） + 10鹰眼（穿透） + 2红月8异变（穿透+暴击）
  • 极致爆发流：
    10祸源（暴击率） + 10隐匿 + 10鹰眼

三、连招技巧#

  1. 基础连招：
    1A2A1A（3级前清野/消耗）
  2. 突进收割：
    31A21A（大招起手定身，接技能穿插平A）
  3. 飞雷神连招：
    31A2A1A3（镜像场内反复换位，持续刷新技能）

四、打野节奏思路#

前期（1-4级）#

  1. 开局红BUFF：
    点二技能快速清野，利用被动刷新CD提升效率
  2. 抢河道之灵：
    1分钟时抵达中路，配合惩戒争夺经济。
  3. 反野策略：
    针对敌方弱势打野（如李白、阿轲）入侵蓝区，压制发育。

中期（4-10级）#

  1. 抓人优先级：
    优先抓发育路，利用大招越塔强杀脆皮。
  2. 控龙时机：
    击杀敌方打野后立即开暴君/主宰，扩大团队优势。

后期（10级后）#

  1. 带线牵制：
    利用镜像快速清兵推塔，迫使敌方回防。
  2. 绕后切C：
    团战第二时间进场，直切射手/法师，连招秒杀后撤离。

五、团战与细节技巧#

  1. 进场时机：
    等待敌方交关键控制技能（如张良大招）后再切入。
  2. 镜像场运用：
    横放大招可封锁路径，竖放便于换位追击残血。
  3. 探草技巧：
    释放二技能命中草丛敌人触发音效，判断埋伏位置。

六、克制与注意事项#

  • 克制英雄：
    鲁班七号、妲己等无位移脆皮。
  • 被克英雄：
    东皇太一（压制）、张良（强控）、兰陵王（隐身偷袭）。
  • 关键细节：
    • 蓝BUFF必拿（镜依赖冷却缩减）
    • 逆风局避免正面团战，专注带线偷塔

​​暃#

一、技能机制解析#

  1. 被动-玉息·星眸
    上墙后获得100点移速加成并探测首个敌人视野（包括隐身单位），核心战略价值在于反制兰陵王、阿轲等隐身英雄。
  2. 一技能-逐玉·锋回
    • 地面形态：直线飞刃两段伤害（类似铠1技能）
    • 墙上形态：突进下墙获得3层护盾+2次强化普攻（攻速提升75%），第二段可击飞敌人
  3. 二技能-踏玉·飞檐
    核心位移技能，可连续跳跃墙体4次（跳跃次数越多，下墙后CD越长），上墙后刷新一技能并降低CD，战略意义大于输出
  4. 三技能-凝玉·之印
    两段斩杀技：第一段标记敌人，第二段对生命值低于**12%**的目标直接斩杀（无视名刀/复活甲），墙上释放时攻击范围扩大至360°

二、出装与铭文推荐（适配5月改版）#

出装方案#

  • 新版核心出装（攻速流）：
    贪婪之噬 → 急速战靴 → 寒霜袭侵 → 冰痕之握 → 破军 → 纯净苍穹
    改版后更依赖墙上强化普攻消耗，寒霜袭侵提供攻速+减速，冰痕之握增强坦度
  • 极限爆发流（顺风局）：
    追击刀锋 → 抵抗之靴 → 暗影战斧 → 宗师之力 → 破军 → 碎星锤

铭文搭配#

  • 通用方案：
    5夺萃+5隐匿（续航+移速） + 10鹰眼（穿透） + 9异变+1红月（破攻速阈值）
  • 暴击流：
    10隐匿 + 10鹰眼 + 3无双+7祸源（后期爆发）

三、连招技巧与实战应用#

基础连招#

  1. 清野/抓人起手：
    2（上墙）→1（下墙）→1（二段击飞）→A→A
    利用强化普攻回血，触发冰痕之握减速留人
  2. 远距离追击：
    2（连续跳跃）→1（下墙）→3（标记）→A→A→3（斩杀）
    类似兰陵王2技能接普攻机制

高阶连招#

  1. 团战收割：
    2（墙体突进）→3（360°标记）→1（群体击飞）→A→A→3（斩杀）
    利用墙上大招范围优势
  2. 新版消耗流：
    2（存储四段位移）→多次跳跃→1（远程强化普攻）→撤回墙体
    改版后新增弹射机制，适合poke战术

四、打野节奏与运营思路#

前期（1-4级）#

  • 开局策略：
    必买打野刀→主升二技能，红BUFF开提升刷野效率（改版后清野速度下降需谨慎）
  • 反野要点：
    针对露娜、韩信等前期弱势打野，利用墙体跳跃实现3秒跨区突袭

中期（4-10级）#

  • 控龙优先级：
    击杀敌方打野后立即开暴君，大招斩杀线可秒杀残血主宰
  • 抓人路线：
    优先发育路→对抗路→中轴，墙体突进速度远超常规打野

后期（10级后）#

  • 带线牵制：
    利用四段位移实现1人带三线战术，遭遇围剿时可瞬间跨越5堵墙体撤离
  • 团战定位：
    第二时间进场，专切射手/法师，新版玉化机制无视名刀直接斩

五、克制关系与细节技巧#

英雄克制#

  • 被克英雄：
    东皇太一（压制）、张良（强控）、金蝉（紧箍限制位移）
  • 天克目标：
    鲁班七号、妲己等无位移脆皮，伽罗（破盾机制反制）

操作细节#

  1. 墙体视野：
    上墙时可探测草丛，但不暴露自身视野（需贴近墙体边缘）
  2. 玉化斩杀：
    新版大招新增玉化标记，被标记目标无法通过治疗/护盾抵消斩杀
  3. 逃生艺术：
    被追击时采用2→1→2→1循环，实现无限墙体马拉松战术

六、版本强度评估#

  • 优势：
    ✔️ 全峡谷最强机动性（四段存储位移）
    ✔️ 战略级视野控制+无视保命装斩杀
    ✔️ 新版弹射机制提升团战AOE能力
  • 弱点：
    ❌ 前期清野速度全打野倒数
    ❌ 怕硬控集火（新增25%受控负面效果）
    ❌ 强化普攻射程削弱50码 虎

裴擒虎#

一、双形态技能核心机制#

  1. 人形态特点

    • 被动「寸劲」：普攻附带叠加法术伤害（3秒内同一目标最高3倍），每次命中回复5点能量
    • 一技能「冲拳式」：直线减速+目标当前生命值8%的法术伤害，命中英雄回复20能量
    • 二技能「气守式」：护盾+持续灼烧+30%攻速提升，护盾存在期间普攻可延长灼烧时间
    • 切换虎形态：获得两次强化普攻（首次带穿墙位移）

  2. 虎形态特点

    • 被动「念气」：增加物攻/双抗/移速
    • 一技能「虎啸式」：斩杀技，目标每损失1%生命增伤1%，击杀目标返还20能量
    • 二技能「虎跃式」：地形折返跳，命中单位返还50%冷却+回复能量
    • 切换人形态：强化普攻附带扇形AOE伤害

二、出装与铭文配置#

版本强势出装#

  • 核心装备：
    贪婪之噬 → 抵抗之靴 → 末世 → 宗师之力 → 碎星锤 → 破军
    • 新版思路：末世触发被动高频伤害，宗师强化形态切换爆发
    • 备选调整：
      • 敌方多控：第三件补纯净苍穹免伤
      • 逆风局：冰痕之握+暴烈之甲提升容错

铭文推荐#

  • 攻速暴击流：
    10狩猎（移速/攻速） + 10鹰眼（穿透） + 5红月5异变（攻速/穿透）
  • 爆发穿透流：
    10隐匿（攻击/移速） + 10鹰眼 + 10异变（极致穿透）

三、连招体系与实战应用#

基础连招#

  1. Gank起手式：
    人形态2技能护盾→1技能减速→切换虎形态2A1A

    • 关键点：护盾持续期间普攻触发灼烧

  2. 地形折返跳：
    虎形态2技能触碰墙体→折返跳回→切换人形态强化AOE

    • 应用场景：龙坑争夺/越塔强杀

高阶连招#

  1. 团战切C：
    人形态远程消耗→虎形态2穿墙突进→1斩杀→切人形态扇形AOE

    • 能量管理：确保切换形态后至少保留40能量

  2. 无伤磨塔：
    人形态2技能护盾→强化普攻→立即撤出塔范围

    • 单次可对防御塔造成1200+伤害

四、打野节奏与运营思路#

前期（1-4级）#

  • 开局策略：

    • 红BUFF开局：利用人形态2技能护盾无伤速刷
    • 1分钟必争：中路河道之灵争夺（虎形态折返跳抢怪）

  • 反野优先级：

    • 针对露娜/阿轲等弱势打野，2级可越墙突袭蓝区

中期（4-10级）#

  • 控龙节奏：

    • 暴君＞主宰：虎形态斩杀线可秒杀12%血量暴君
    • 偷龙技巧：人形态远程消耗+虎形态折返跳出龙坑

  • 推塔策略：

    • 优先推发育路，利用人形态强化普攻远程磨塔

后期（10级后）#

  • 带线牵制：
    • 四段位移实现「三线分推」，遭遇围剿可穿5墙撤离
    • 关键提示：带线时保持人形态，遭遇战秒切虎形态反打

五、团战细节与克制关系#

  1. 进场时机：

    • 第二时间切入，专盯交完闪现的射手/法师
    • 地形利用：野区团战强度提升30%（折返跳空间大）

  2. 能量管理：

    • 人形态多用1技能回能量，虎形态确保2技能命中回能

  3. 克制关系：

    • 天敌：东皇太一（压制）、金蝉（限制位移）
    • 猎物：鲁班七号、妲己等无位移脆皮

六、版本强度评估（2025年5月）#

  • 优势：
    ✔️ 前中期野区统治力TOP3
    ✔️ 双形态适应全期作战节奏
    ✔️ 混伤机制克制单抗性阵容
  • 弱点：
    ❌ 后期团战容错率低
    ❌ 能量断档期易被反杀

韩信#

一、技能机制与核心玩法#

1. 技能解析#

  • 被动「杀意之枪」
    每四次普攻触发击飞0.5秒，技能命中目标增加50%攻速。空A三次存被动，突进后强化普攻可瞬间打出击飞+高额伤害。
  • 一技能「无情冲锋」
    两段位移（第一段击飞），CD短，核心突进/逃生技能。反野时穿墙抢BUFF，团战绕后切C位。
  • 二技能「背水一战」
    后跳+强化普攻横扫，命中返还50%冷却。反向释放调整位置，配合一技能实现“折返跳”拉扯。
  • 三技能「国士无双」
    霸体+免伤30%，持续横扫造成多段伤害。注意释放期间可被硬控打断，需预判敌方技能。

2. 核心打法#

  • 定位：高机动刺客，主打带线牵制与后排收割。
  • 节奏关键：前期速四反野，中期控龙分推，后期绕后切C。

二、出装与铭文推荐#

1. 出装方案#

  • 暴击穿透流（当前主流）：
    贪婪之噬 → 抵抗之靴 → 暗影战斧 → 无尽战刃 → 宗师之力 → 破军
    • 核心逻辑：暗影战斧提供穿透和冷却缩减，无尽+宗师提升暴击爆发，破军强化收割能力。
    • 备选调整：
      • 敌方多控：中期补纯净苍穹免伤
      • 逆风局：冰痕之握+暴烈之甲提升容错

2. 铭文搭配#

  • 通用暴击流：
    10夺萃（续航） + 10鹰眼（穿透） + 7祸源3无双（暴击率+暴击效果）
  • 极致穿透流：
    10隐匿（攻击/移速） + 10鹰眼 + 10异变（极致穿透）

三、连招技巧与实战应用#

1. 基础连招#

  • Gank起手式：
    1技能第一段（击飞）→ 普攻触发被动 → 1技能第二段追击 → 2技能横扫 → 大招收割
  • 反野逃生：
    2技能反向跳墙 → 1技能穿墙 → 普攻接大招免伤

2. 高阶连招#

  • 团战切C：
    存被动→2技能突进→横扫→1技能击飞→大招霸体抗控→闪现离场
  • 无伤磨塔：
    1技能穿墙进塔→普攻→2技能反向撤出（单次可造成1200+伤害）

四、打野节奏与运营思路#

1. 前期（1-4级）#

  • 开局策略：
    • 红BUFF开局：利用人形态2技能护盾无伤速刷
    • 1分钟必争：中路河道之灵争夺（虎形态折返跳抢怪）
  • 反野优先级：
    针对露娜/阿轲等弱势打野，2级可越墙突袭蓝区

2. 中期（4-10级）#

  • 控龙节奏：
    • 暴君＞主宰：利用大招免伤强开龙团
    • 带线牵制：优先推发育路，三路分推迫使敌方回防
  • 抓人路线：
    优先发育路→对抗路→中轴，利用多段位移快速支援

3. 后期（10级后）#

  • 团战定位：
    第二时间进场，专盯交完闪现的射手/法师
  • 逆风策略：
    无限带线牵制，利用三段位移实现“一拖三”战术

五、克制关系与操作细节#

1. 英雄克制#

  • 天敌：
    东皇太一（压制）、张良（强控）、金蝉（限制位移）
  • 猎物：
    鲁班七号、妲己等无位移脆皮，伽罗（破盾机制反制）

2. 细节技巧#

  • 能量管理：
    人形态多用1技能回能量，虎形态确保2技能命中回能
  • 探草技巧：
    提前空A三次存被动，突进草丛瞬间触发击飞
  • 地形利用：
    野区作战强度提升30%，多利用墙体折返跳拉扯

六、版本强度评估#

  • 优势：
    ✔️ 全峡谷最强机动性（三段位移+墙体穿越）
    ✔️ 前中期野区压制力TOP3
    ✔️ 混伤机制克制单抗性阵容
  • 弱点：
    ❌ 后期团战容错率低
    ❌ 依赖蓝BUFF续航（建议常带蓝buff）

火舞#

一、技能机制与核心玩法#

1. 技能解析与操作技巧#

  • 被动「忍蜂」
    每次释放技能后可翻滚位移，脱离战斗后首次普攻附带击退效果。核心技巧：
    • 空A三次存被动，突进后强化普攻可瞬间打出击飞+高额伤害
    • 设置中将「自动追击距离」调整为近距离，避免被动被野怪意外触发
  • 一技能「飞翔龙炎阵」
    突进击飞技，CD短且可穿墙。建议搭配闪现实现超远距离开团（如一闪接大招）
  • 二技能「花蝶扇」
    核心消耗技，命中后降低目标90%移速和200点法抗。设置优化：
    • 开启「追踪轮盘施法」自动瞄准，提高命中率
    • 配合被动翻滚调整站位，规避敌方前排阻挡
  • 三技能「必杀·忍蜂」
    高爆发突进技，可击退路径上所有敌人。连招关键：
    • 大招后接闪现可调整击退方向（如折返大闪）
    • 优先攻击被二技能减抗的目标提升斩杀线

2. 战场定位#

  • 前期：远程消耗型法师，利用二技能压制中路
  • 中期：刺客型法核，通过多段位移切入后排
  • 后期：团战收割者，需把握进场时机避免被集火

二、出装与铭文推荐#

1. 版本强势出装#

  • 爆发秒杀流（主流选择）：
    噬神之书 → 抵抗之靴 → 回响之杖 → 博学者之怒 → 虚无法杖 → 辉月
    • 核心逻辑：回响+帽子组合提升中期爆发，辉月规避关键控制
    • 备选调整：
      • 敌方多回复：中期补梦魇之牙
      • 逆风局：时之预言替代辉月提升容错

2. 铭文搭配#

  • 通用穿透流：
    10梦魇（法强/穿透） + 10心眼（攻速/穿透） + 10贪婪（续航）
  • 极致机动流：
    10梦魇 + 10怜悯（冷却缩减） + 5狩猎5贪婪（移速/续航）

三、连招体系与实战应用#

1. 基础连招#

  • 蹲草瞬杀：
    2（自动瞄准）→ 强化普攻 → 3（击退）→ 1（追击）→ 2（收尾）
    • 可在1.5秒内打出4000+爆发伤害
  • 反打连招：
    被动击退 → 2（减抗） → 1（控制） → 3（收割）
    • 专克李白/韩信等突进型刺客

2. 高阶连招#

  • 团战开团技：
    闪现 → 3（群体击退） → 辉月（规避伤害） → 2（减速） → 1（补控）
    • 需预判敌方C位站位，可配合鬼谷子等强控英雄
  • 极限逃生技：
    2（减速追击者） → 被动翻滚穿墙 → 1（二次位移）

四、节奏把控与运营思路#

1. 前期对线（1-4级）#

  • 清线技巧：
    • 侧向释放二技能穿透兵线，搭配回响被动快速清兵
    • 利用河蟹叠被动，2分钟时存3次强化普攻争夺暴君
  • 游走策略：
    • 优先支援发育路，通过墙体翻滚实现「无视野Gank」

2. 中期团战（4-10级）#

  • 控龙优先级：
    • 黑暗暴君 > 主宰，利用二技能减抗提升rush速度
    • 偷龙技巧：墙体翻滚进龙坑 → 大招击退 → 辉月规避惩戒
  • 带线牵制：
    • 通过「三线分推」迫使敌方回防，注意观察小地图预判围剿

3. 后期决胜（10级后）#

  • 团战定位：
    • 第二时间进场，专盯交完保命技能的射手（如马可波罗净化CD）
    • 通过「二闪」或「大闪」改变技能轨迹，出其不意切C
  • 逆风翻盘：
    • 利用超远距离二技能消耗守塔，经济差5000+时仍具备秒C能力

五、克制关系与细节优化#

1. 英雄克制#

  • 天敌：
    东皇太一（压制）、金蝉（紧箍限制位移）、盾山（格挡飞行物）
  • 猎物：
    鲁班七号（无位移）、王昭君（技能前摇明显）、伽罗（破盾机制反制）

2. 操作细节#

    • 二技能命中后获得3秒敌方视野，可破解兰陵王隐身
  • 能量管理：
    • 每次技能命中回复25能量，连招间隙穿插普攻维持续航

六、版本强度评估#

马可#

一、技能机制与核心玩法#

1. 技能解析与操作技巧#

  • 被动「连锁反应」
    普攻与技能命中叠加印记，叠满10层触发5秒真实伤害。
    关键细节：

    • 对线期优先用1技能消耗叠加印记，触发真伤后配合大招可秒杀脆皮
    • 设置「自动追击距离」为近战模式，避免被野怪/小兵干扰印记叠加

  • 一技能「华丽左轮」
    核心消耗技，子弹数量随攻速提升（最高11发）。
    操作要点：

    • 侧向45°角释放穿透兵线，提升清线效率
    • 攻速达到150%时（末世+电刀+攻速鞋）达到最大输出频率

  • 二技能「漫游之枪」
    位移+探草+20%伤害增益三合一技能。
    进阶技巧：

    • 500码内有敌人时触发预警特效，可探测草丛埋伏
    • 反向跳跃规避钟馗钩子、吕布1技能等直线控制

  • 三技能「狂热弹幕」
    高爆发AOE技能，攻速影响弹幕次数。
    使用策略：

    • 进场前确保触发被动真伤，搭配晕眩实现群体控制链
    • 被集火时可用作穿墙逃生（如躲避兰陵王追击）

二、出装与铭文推荐#

1. 版本主流出装方案#

  • 暴击穿透流（通用选择）：
    急速战靴 → 末世 → 闪电匕首 → 冰霜冲击 → 破晓 → 魔女斗篷

    • 核心逻辑：末世+电刀快速达成150%攻速阈值，冰霜冲击强化控制
    • 备选调整：
      • 敌方多回复阵容：中期补制裁之刃替代电刀
      • 逆风局：不祥征兆替代冰霜冲击提升生存

  • 半肉持续输出流（新手友好）：
    护甲鞋 → 末世 → 寒霜侵袭 → 不祥征兆 → 魔女斗篷 → 破晓
    特点：血量8500+，适合拉扯消耗打法

2. 铭文搭配#

  • 攻速暴击流：
    10红月（攻速/暴击） + 10鹰眼（穿透） + 10狩猎（移速/攻速）
  • 半肉续航流：
    10宿命（攻速/血量） + 10鹰眼 + 10夺萃（物理吸血）

三、连招体系与实战应用#

1. 基础连招#

  • 对线消耗连招：

    伤害峰值：4级时可达2000+真实伤害

  • 爆发收割连招：
    存被动 → 2技能突进 → 1技能扫射 → 大招进场 → 晕眩控场
    应用场景：敌方脆皮站位集中时使用

2. 高阶技巧#

  • 龙坑双控：
    利用墙体卡视野，2技能穿墙接大招+冰霜冲击实现双重控制抢龙
  • 三线分推：
    2技能+大招连续穿墙带线，迫使敌方分散防守
