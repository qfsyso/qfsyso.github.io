更换 iRedMail 的域名#

（如从 mail.example.com 更换为 mail.mydomain.com）， 需要修改多个配置文件和数据库记录，并确保新的域名配置正确。 以下是操作步骤：

步骤 1：更新 DNS 记录#

在域名托管平台更新 mail.mydomain.com 的 DNS 记录： A记录：指向邮件服务器的 IP 地址。 MX记录：指向 mail.mydomain.com。 SPF 记录：确保包括新的域名。 DKIM 和 DMARC 记录：重新生成并更新。 确保 DNS 更改后可以通过 dig 或 nslookup 验证生效。

步骤 2：更新主机名#

修改主机名：

 
hostnamectl set-hostname mail.mydomain.com

更新 /etc/hosts 文件：

127.0.0.1   localhost
127.0.1.1   mail.mydomain.com mail

更新 /etc/mailname 文件：

mail.mydomain.com

步骤 3：更新 iRedMail 配置文件#

修改 Postfix 配置： 编辑 /etc/postfix/main.cf：


myhostname = mail.mydomain.com
mydomain = mydomain.com
myorigin = $mydomain

重启 Postfix：

systemctl restart postfix

修改 Dovecot 配置： 编辑 /etc/dovecot/dovecot.conf 和 /etc/dovecot/conf.d/10-mail.conf，确保更新主机名：

ssl_cert = </etc/ssl/certs/mail.igrmail.pics.crt
ssl_key = </etc/ssl/private/mail.igrmail.pics.key

重启 Dovecot：

systemctl restart dovecot

修改 Amavisd 配置： 编辑 /etc/amavis/conf.d/50-user：

$myhostname = 'mail.mydomain.com';

重启 Amavisd：

 
systemctl restart amavis

修改 iRedAdmin 配置： 编辑 /opt/www/iredadmin/settings.py：

web_hostname = 'mail.igrmail.pics'

重启 iRedAdmin：

systemctl restart iredadmin

步骤 4：更新数据库记录#

使用以下命令登录数据库（通常是 MySQL/MariaDB）：

mysql -u root -p

更新 vmail 数据库中的域名：

 
USE vmail;
UPDATE domain SET domain='new' WHERE domain='old';
UPDATE alias SET address=REPLACE(address, 'old', '@new');
UPDATE mailbox SET username=REPLACE(username, '@old', '@new');

步骤 5：重新生成 SSL 证书#

如果你使用 Let’s Encrypt：

 
certbot --apache -d mail.xxx

将新生成的证书路径更新到相关服务的配置文件中（如 Postfix、Dovecot）。

步骤 6：重启所有相关服务#

确保所有服务都重启以加载新的配置：

 
systemctl restart postfix dovecot amavis clamav spamassassin iredapd iredadmin

步骤 7：验证配置#

检查服务状态：

systemctl status postfix dovecot amavis

发送和接收测试邮件，确保新域名能正常工作。 验证 MX、SPF、DKIM 和 DMARC 记录是否正确。 完成这些步骤后，你的邮件服务器将会切换到新的域名。

debian 安装 MariaDB#

在Debian系统上安装MariaDB，你可以按照以下步骤操作：

更新系统包列表：

sudo apt update 安装MariaDB服务器：

sudo apt install mariadb-server 安全设置MariaDB，包括设置root密码和删除匿名用户：

sudo mysql_secure_installation 启动MariaDB服务：

sudo systemctl start mariadb 使MariaDB服务在系统启动时自动运行：

sudo systemctl enable mariadb 登录MariaDB以确保一切正常：

sudo mysql -u root -p 当系统提示输入密码时，输入你在安全设置步骤中设置的密码。

以上步骤将在Debian系统上安装MariaDB，并确保其安全和自动启动。

连接MariaDB (like mysql) SHOW DATABASES; SHOW TABLES;
