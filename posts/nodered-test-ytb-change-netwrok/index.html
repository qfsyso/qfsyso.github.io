Setup Node-RED#

一、前提条件：安装 Node.js#

Node-RED 需要 Node.js 环境，需先安装 Node.js。

  1. 打开 Node.js 官网：https://nodejs.org/
  2. 下载并安装 LTS（长期支持）版本
  3. 安装完成后打开终端（Win + X → 终端），输入以下命令检查是否安装成功：
node -v
npm -v

二、全局安装 Node-RED#

在命令行中执行以下命令安装 Node-RED：

npm install -g --unsafe-perm node-red

说明：

  • -g：表示全局安装
  • --unsafe-perm：解决某些系统中权限问题

三、启动 Node-RED#

安装完成后，在终端中运行：

node-red

输出类似以下内容：

Server now running at http://127.0.0.1:1880/

四、访问 Node-RED 编辑器#

打开浏览器访问：

http://localhost:1880

即可进入图形化流程编辑界面。

🛠️ 五、可选：创建桌面快捷方式（.bat 启动脚本）#

  1. 创建一个文本文件，内容如下：

    @echo off
title Node-RED
node-red
pause

  2. 保存为 node-red.bat

  3. 双击即可启动 Node-RED

六、（可选）将 Node-RED 作为后台服务运行#

你可以使用如下工具实现后台服务：

  • pm2（推荐）：适用于 Node.js 程序管理
  • nssm（适用于 Windows）：https://nssm.cc/

常见问题#

  • 如果 npm install 失败，请确认网络是否畅通，或使用淘宝镜像源：

    npm config set registry https://registry.npmmirror.com

puppeteer-browser-launch

headless 设为 true（无界面运行），如果想看真机效果可设为 false。

executablePath 一般留空，默认使用内置 Chromium。如果你有本地 Chrome 路径，也可以填入。

puppeteer-page-goto

url 填 对应

waitUntil 设为 networkidle2，保证页面资源加载完毕。

puppeteer-page-click

selector 填目标 CSS 选择器：.proxies-speed-test

delay 是点击前的延迟（ms），可根据需要调整。

puppeteer-browser-close

完成后关闭浏览器实例，防止内存泄漏。
