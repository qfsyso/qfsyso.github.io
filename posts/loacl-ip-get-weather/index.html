获取经纬度接口#

获取当前网络经纬度的免费且稳定的接口有几个选择。这些接口通常提供基于IP地址的服务，能够返回访问者的大致地理位置信息，包括经纬度。请注意，基于IP地址的服务精度有限，特别是在城市范围内，因为它们依赖于ISP提供的数据。以下是一些常用的免费API选项：

1. ip-api.com#

URL: http://ip-api.com/json 方法: GET 返回值: JSON格式的数据，包含国家、地区、城市、邮编、经纬度等信息。

2. ipinfo.io#

URL: https://ipinfo.io/json 方法: GET 返回值: JSON格式的数据，包括IP地址、城市、区域、国家、位置（经纬度）、组织等信息。

3. freegeoip.app#

URL: https://api.freegeoip.app/json/ 方法: GET 请求头: 需要加入 Accept-Language: en 和 User-Agent: YourAppName，其中 YourAppName 应用的名字。 返回值: JSON格式的数据，包括IP、国家代码、国家名、地区、城市、邮政编码、纬度、经度等信息。

使用示例


<!DOCTYPE html>
<html>
<head>
    <title>Get Current Position</title>
</head>
<body>
    <button onclick="getLocation()">Get Location</button>
    <p id="location"></p>

    <script>
        function getLocation() {
            fetch('https://ipinfo.io/json')
                .then(response => response.json())
                .then(data => {
                    const location = document.getElementById('location');
                    location.textContent = `Latitude: ${data.loc.split(',')[0]}, Longitude: ${data.loc.split(',')[1]}`;
                })
                .catch(error => console.error('Error:', error));
        }
    </script>
</body>
</html>

两者结合

            display: flex;

        }
    </style>

<body>
    <button onclick="getWeather()">Get Weather</button>
    <div id="weather-info" class="section"></div>

    <script>
        function getWeather() {
            // 获取经纬度
            fetch('https://ipinfo.io/json')
                .then(response => response.json())
                .then(data => {
                    const lat = data.loc.split(',')[0];
                    const lon = data.loc.split(',')[1];
                    
                    // 使用经纬度调用天气API
                    return fetch(`https://weather.sl.al/?lat=${lat}&lon=${lon}`);
                })
                .then(response => response.json())
                .then(weatherData => {
                    const weatherInfo = document.getElementById('weather-info');
                    weatherInfo.innerHTML = '';

                    // 显示当前天气
                    const currentWeather = `
                        <div class="section">
                            <div class="header">Current Weather in ${weatherData.location.city}</div>
                            <div class="details">
                                Temperature: ${weatherData.current.temperature}°C<br>
                                Feels Like: ${weatherData.current.feelsLike}°C<br>
                                Description: ${weatherData.current.description}<br>
                                Air Quality: ${weatherData.current.airQuality.category}<br>
                                ${weatherData.current.airQuality.statement}
                            </div>
                        </div>
                    `;
                    weatherInfo.innerHTML += currentWeather;

                    // 显示今日天气详情
                    const todayWeather = `
                        <div class="section">
                            <div class="header">Today's Weather (${weatherData.daily[0].date})</div>
                            <div class="details">
                                High: ${weatherData.daily[0].high}°C<br>
                                Low: ${weatherData.daily[0].low}°C<br>
                                Day: ${weatherData.daily[0].dayDescription}<br>
                                Night: ${weatherData.daily[0].nightDescription}<br>
                                Rain Probability: ${weatherData.daily[0].rainProbability}
                            </div>
                        </div>
                    `;
                    weatherInfo.innerHTML += todayWeather;

                    // 显示未来几天的天气预报
                    const dailyForecast = `
                        <div class="section">
                            <div class="header">Daily Forecast</div>
                            <div class="daily-forecast">
                                ${weatherData.daily.slice(1).map(day => `
                                    <div class="daily-item">
                                        Date: ${day.date}<br>
                                        High: ${day.high}°C<br>
                                        Low: ${day.low}°C<br>
                                        Day: ${day.dayDescription}<br>
                                        Night: ${day.nightDescription}<br>
                                        Rain Probability: ${day.rainProbability}
                                    </div>
                                `).join('')}
                            </div>
                        </div>
                    `;
                    weatherInfo.innerHTML += dailyForecast;

                    // 显示每小时的天气预报
                    const hourlyForecast = `
                        <div class="section">
                            <div class="header">Hourly Forecast</div>
                            <div class="hourly-forecast">
                                ${weatherData.hourly.map(hour => `
                                    <div class="hourly-item">
                                        Time: ${hour.time}<br>
                                        Temperature: ${hour.temperature}°C<br>
                                        Rain Probability: ${hour.rainProbability}
                                    </div>
                                `).join('')}
                            </div>
                        </div>
                    `;
                    weatherInfo.innerHTML += hourlyForecast;
                })
                .catch(error => console.error('Error:', error));
        }
    </script>
</body>
</html>

结果为当前天气、今日天气详情、未来几天的天气预报以及每小时的天气预报~ （Current Weather、Today’s Weather、Daily Forecast、Hourly Forecast）
