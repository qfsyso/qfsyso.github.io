tvbox config#

如 https://cj.lziapi.com/api.php/provide/vod

获取 JSON 格式数据：

https://cj.lziapi.com/api.php/provide/vod/at/json

获取 XML 格式数据：

https://cj.lziapi.com/api.php/provide/vod/at/xml

指定来自 lzm3u8 的资源，并获取 XML：

https://cj.lziapi.com/api.php/provide/vod/from/lzm3u8/at/xml/
